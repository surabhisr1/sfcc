A Progressive Web App (PWA) created with Mobify Platform is compatible with the following web browsers:

### Mobile devices

| Operating system | Browser          |
|------------------|------------------|
| Android 4.4.4+   | Chrome 53+       |
| Android 4.4.4+   | Samsung Internet |
| iOS 9.x+         | Chrome 53+       |
| iOS 9.x+         | Safari 9+        |

### Tablets

| Operating system | Browser          |
|------------------|------------------|
| Android 4.4.4+   | Chrome 53+       |
| Android 4.4.4+   | Samsung Internet |
| iOS 9.x+         | Chrome 53+       |
| iOS 9.x+         | Safari 9+        |


### Desktops and laptops

| Operating system | Browser              |
|------------------|----------------------|
| macOS 10.x       | Safari 10+           |
| macOS 10.x       | Firefox 49+          |
| Windows 8.1      | Internet Explorer 11 |
| Windows 10.x     | Chrome 53+           |
| Windows 10.x     | Edge 14+             |
| Windows 10.x     | Firefox 49+          |
| Windows 10.x     | Internet Explorer 11 |
