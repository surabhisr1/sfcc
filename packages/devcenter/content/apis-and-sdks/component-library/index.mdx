The Mobify Platform includes an extensive library of React components for creating your storefront. Each component has been optimized for responsive design and supports best practices for usability and accessibility.

We also offer a Sketch file that includes symbols for creating mockups using our components. See our guide to [Using the Mobify UI Kit](../how-to-guides/categories/design/mobify-ui-kit) for more information.

Select a component or template from the sidebar navigation to access props tables, import statements, code examples, and design considerations.

If you edit the source code below, the example component will re-render to reflect your edits.
