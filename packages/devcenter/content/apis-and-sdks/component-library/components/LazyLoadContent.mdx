### Basic example

```jsx react-live=true
<LazyLoadContent placeholder={<SkeletonBlock height="250px" width="250px" />}>
    <Image
        src="https://i.pinimg.com/564x/af/f7/43/aff743479fc29789d2231329453b1abc.jpg"
        alt="Alt text"
        height="250px"
        width="250px"
    />
</LazyLoadContent>
```

### Threshold and scrollCheckInterval props

These props allow you to delay the loading of an image until it scrolls within a certain range of the viewport.

```jsx react-live=true
<div style={{paddingTop: '333px'}}>
    <LazyLoadContent
        placeholder={<SkeletonBlock height="250px" width="250px" />}
        threshold={200}
        scrollCheckInterval={400}
    >
        <Image
            src="https://i.pinimg.com/564x/af/f7/43/aff743479fc29789d2231329453b1abc.jpg"
            alt="Alt text"
            height="250px"
            width="250px"
        />
    </LazyLoadContent>
</div>
```
