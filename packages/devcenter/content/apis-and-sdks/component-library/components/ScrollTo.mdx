<div class="component-intro">

A wrapper component for [Button](Button) that allows you to scroll to different points on the page. This component can be passed either a page height (in pixels) or an element selector as its target. If multiple elements are found with the target selector, it will scroll to the first one returned by `querySelectorAll()`.

</div>

### Basic example

Scroll to the top of the page (target height of 0) with custom duration.

```jsx react-live=true
<ScrollTo target={0} duration={200}>
    <Button>Click to go to the top of the page</Button>
</ScrollTo>
```
