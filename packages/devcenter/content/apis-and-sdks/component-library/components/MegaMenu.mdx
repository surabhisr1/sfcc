<div class="component-intro">

The `MegaMenu` component is responsible for rendering a tree of `MegaMenuItem`s that users
can traverse the navigation tree. The `MegaMenu` can first render all of the root-level items. The user may hover over these items using their mouse or tap to view the second level of items, at which point the `MegaMenu` can reveal those menu items.

`MegaMenu` cannot be used by itself. It must be inserted into a `Nav` component.

In order to remain accessible to search engine crawlers, the Mega Menu family of components is designed so that it always renders the navigation in a way that can be read and understood by those crawlers. The Mega Menu may not be entirely visible to a human user, but it's available to search engine crawlers. In this way, the Mega Menu also acts as a sort of sitemap.

Related components: [Nav](Nav) and [MegaMenuItem](MegaMenuItem).

### Basic example

The example below demonstrates the bare minimum code needed to render the `MegaMenu` component. You will notice that it's not fully functional-- clicking the items does nothing. The other examples below will build upon this first example to add functionality.

```jsx react-live=true
<Nav
    root={{
        title: 'Phone Book',
        path: '/',
        children: [{title: 'A', path: '/a/'}, {title: 'B', path: '/b/'}, {title: 'C', path: '/c/'}]
    }}
>
    <MegaMenu />
</Nav>
```

Functionality is introduced by passing in both an `onPathChange` callback and the current active path. (For a full explanation, refer to the [`Nav` component's documentation](Nav). Additionally we add some children items to allow further traversal:

```jsx react-live=true
class StatefulExample extends React.Component {
    constructor() {
        this.state = {
            path: '/'
        }
    }

    changePath(path, isLeaf, trigger) {
        // Track the newly active path in the state so the
        // component can highlight it in the UI.
        this.setState({path: path})

        // Only "navigate" anywhere if the trigger was a click!
        if (trigger === 'click') {
            // You might do something like this for leafs:
            // `window.location.href = path`
            window.alert('Navigate to ' + path)
        }
    }

    render() {
        const root = {
            title: 'Phone Book',
            path: '/',
            children: [
                {
                    title: 'A',
                    path: '/a/',
                    children: [{title: 'Alexa', path: '/a/alexa'}]
                },
                {
                    title: 'B',
                    path: '/b/',
                    children: [{title: 'Billy', path: '/a/billy'}]
                },
                {
                    title: 'C',
                    path: '/c/',
                    children: [{title: 'Cassandra', path: '/a/cassandra'}]
                }
            ]
        }

        return (
            <Nav root={root} path={this.state.path} onPathChange={this.changePath.bind(this)}>
                <MegaMenu />
            </Nav>
        )
    }
}
```

### Item factory

It's common for developers to customize how the navigation items are rendered. This can be done using the `MegaMenu` component's `itemFactory` callback.

For example:

```javascript static
const itemFactory = (type, props) => {
    switch (type) {
        case 'custom-1':
            return <CustomItem1 {...props} />
        case 'custom-2':
            return <CustomItem2 {...props} />
        case 'custom-3':
            return <CustomItem3 {...props} />
    }
    return <MegaMenuItem {...props} />
}
```

The callback provides two arguments: `type` and `props`. These arguments come from the Nav component's root prop object. (For more information, see the [Nav documentation](Nav)).

                )
            }

            if (type === 'custom-type') {
                return <CustomMegaMenuItem {...props} />
            }

            return <MegaMenuItem {...props} />
        }

        return (
            <Nav root={root} path={this.state.path} onPathChange={this.changePath.bind(this)}>
                <MegaMenu itemFactory={itemFactory} />
            </Nav>
        )
    }
}
