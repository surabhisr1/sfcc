### Potential uses

-   In the header bar, to allow multiple actions to fit within a small space.
-   Anywhere an icon is used instead of text, and when users may struggle to comprehend the meaning of the icon.

### Accessibility

-   Ensure the label size is small yet legible. Ensure the text size is at least 8px.
-   Because of the small text size, be extra careful with the color contrast between the label and the background color. [Use this handy tool](http://www.contrastchecker.com) to test contrast.

### Best practices

-   Research has shown that using labels with icons can improve comprehension and interactivity.
-   Keep labels short. Icons are designed to occupy a small space and a long label will negate this benefit.
-   Users learn the meaning of an icon through repeat exposure. Consider removing labels once the icon has been seen several times. A good example is the fixed header bar, where labels are removed from the icons after the user starts scrolling.
-   Don't use labels when multiple instances of the icon exist in the same place. For example, no label is required for the + icons in an accordion.
