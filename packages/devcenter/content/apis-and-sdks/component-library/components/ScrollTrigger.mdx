### Basic example

```jsx react-live=true
class StatefulExample extends React.Component {
    constructor() {
        this.state = {
            inView: false
        }
    }

    render() {
        return (
            <div>
                <div style={{height: '200px'}}>
                    {this.state.inView ? 'The cat is visible!' : 'The cat is not visible.'}
                </div>
                <ScrollTrigger
                    onEnter={() => this.setState({inView: true})}
                    onLeave={() => this.setState({inView: false})}
                >
                    <Image
src="https://s-media-cache-ak0.pinimg.com/564x/72/4b/6d/724b6dbf91c378a53d6890bb525c1aa9.jpg"
                        alt="Cat"
                    />
                </ScrollTrigger>
            </div>
        )
    }
}
```

### With offset

A negative `offsetTop` can be used to do something before the content has been scrolled into view:

```jsx react-live=true
class StatefulExample extends React.Component {
    constructor() {
        this.state = {
            inView: false
        }
    }

    render() {
        return (
            <div>
                <div style={{height: '200px'}}>
                    {this.state.inView ? 'The cat is visible!' : 'The cat is not visible.'}
                </div>
                <ScrollTrigger
                    onEnter={() => this.setState({inView: true})}
                    onLeave={() => this.setState({inView: false})}
                    offsetTop={-100}
                >
                    <Image
src="https://s-media-cache-ak0.pinimg.com/564x/72/4b/6d/724b6dbf91c378a53d6890bb525c1aa9.jpg"
                        alt="Cat"
                    />
                </ScrollTrigger>
            </div>
        )
    }
}
```
