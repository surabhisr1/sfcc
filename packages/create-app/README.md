# Generator

## `@mobify/create-app`

### What is it?

This is Mobify's project generator - it's a CLI utility that generates a Mobify PWA
with a few customized settings.

To use it, follow the wizard after running:

```bash
npx @mobify/create-app
```

For help, run:

```bash
npx @mobify/create-app --help
```
